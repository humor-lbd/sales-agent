# 玖森百立业务场景数据

本数据集基于“咪婴伴侣”母婴 O2O / 母婴商城 / 直播电商 / 同城服务业务构造，文件为：

- `app/db/sql/jiusen_scenario_coverage.sql`
- `scripts/load_jiusen_scenario_data.py`

执行方式：

```powershell
$env:PYTHONUTF8='1'
E:\develop\python3.13.2\python.exe scripts\load_jiusen_scenario_data.py
```

只校验：

```powershell
$env:PYTHONUTF8='1'
E:\develop\python3.13.2\python.exe scripts\load_jiusen_scenario_data.py --verify-only
```

## 业务映射

项目工具层只允许四个大区名称，因此按当前系统口径映射：

- 华北区：山西太原主阵地，覆盖产后修复、月嫂到家、陪诊、健康咨询。
- 华东区：母婴商城和直播转化合作城市。
- 华南区：直播电商、母婴用品和课程试点。
- 西南区：服务试点区，并构造近期订单下滑用于异常检测。

## 覆盖范围

新增数据覆盖：

- 订单明细查询：`ORD-JS-CM-*`、`ORD-JS-RF-*` 覆盖完成、退款、取消状态。
- 销售额汇总：本月四个大区都有完成订单。
- 销售员排名：新增 9100 段销售/服务人员，包含经理、顾问、总监。
- 大区排名：华北、华东、华南、西南都有本月订单。
- 产品排名：母婴服务、产康服务、母婴用品、陪诊、课程、直播电商均有销售。
- 环比：`ORD-JS-MOM-*` 覆盖默认等长对比周期。
- 同比：`ORD-JS-YOY-*` 覆盖去年同期。
- 月度趋势：`ORD-JS-TR-*` 覆盖近 6 个月趋势。
- 图表：品类、产品、销售员、大区维度都有可聚合数据。
- 异常检测：覆盖大区订单骤降、产品连续零销售、退单率异常、销售员业绩骤降。

## 典型验证问句

- `本月全公司销售额是多少？`
- `本月华北区销售额是多少？`
- `本月大区业绩排名是什么？`
- `本月产品Top5是什么？`
- `本月销售员Top5是谁？`
- `本月销售额环比怎么样？`
- `本月销售额同比去年怎么样？`
- `近6个月销售趋势是什么？`
- `帮我生成本月按品类的销售额饼图`
- `最近有没有销售异常？`

## 数据命名约定

- 新增销售员 ID：`9101` 到 `9115`
- 新增产品 SKU：`SKU-JS-*`
- 新增订单号：`ORD-JS-*`

这些前缀便于和原始 demo 数据、泛化场景数据区分，也便于重复执行和后续清查。
